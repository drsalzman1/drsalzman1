Sudoku Tutor

Introduction: This app teaches you to solve Sudoku puzzles using the Nishio (trial and error) method. Each puzzle has 81 "squares" arranged in 9 "rows", 9 "columns" and 9 3x3 "boxes". Each digit from 1 to 9 appears exactly once in each row, column and box.

When ready, change the entry mode to "Copy Squares".
Copy Squares: Copy the pre-filled squares from the puzzle you want to solve. To copy a square, select (touch) a destination square then enter (touch) its source number on the keypad. To erase a square, select a square then enter its number on the keypad.

When done, change the entry mode to "Enter Marks".
Enter Marks: Mark the possible values for each empty square. To enter marks, select a square then enter the numbers that don't yet appear in the square's row, column or box. To erase marks, select a square then enter the numbers of the marks to be erased.

When done, change the entry mode to "Enter Squares".
Enter Squares: Enter the value of each empty square. To enter a square, select the square, enter a value, and update any affected marks. If you're unsure of a value, touch "Save" before entering the trial value. If you hit a dead end, touch "Recall".

Note that multiple save levels are supported.
